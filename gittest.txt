github 와 연동하기

1. github 에 저장소 생성하기
2. 저장소 주소를 복사하기
3. 소스트리에서 clone 버튼 클릭
4. clone 화면의 원격 저장소 주소 입력란에 생성한 github 저장소 주소 붙여넣기
5. 로컬 컴퓨터에 저장할 위치 선정하기
6. clone 버튼 클릭하기

원격 설정
1. 설정 버튼 클릭
2. '원격 저장소'에 '편집' 클릭
3. url 정보 중 'https://' 뒤에 github에서 발급한 '토큰정보' 에 '@github.com/본인아이디/저장소명.git' 으로 수정
4. '확인' 버튼 클릭하여 설정 완료
5. 파일 수정 후 '커밋'을 진행
6. 'push' 버튼을 클릭하여 업로드 확인.
